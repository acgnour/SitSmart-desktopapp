# SitSmart Desktop Productivity Timer

SitSmart is a modern desktop productivity application built with Python and PyQt5.
The application helps users stay focused and maintain healthy work habits by using a customizable focus timer inspired by the Pomodoro technique.

---

## Features

- Modern dark-mode desktop UI
- Custom work and break timer inputs
- Start, pause, and reset controls
- Interactive countdown timer
- Desktop notification popup when session ends
- Sound alert using system beep
- Responsive PyQt5 layouts
- Gradient buttons and modern illustration design

---

## Technologies Used

- Python 3
- PyQt5

## How to Run

### Install PyQt5

```bash
pip3 install pyqt5
```

### Run the Application

```bash
python3 main.py
```

---

## Purpose of the Project

The goal of SitSmart is to create a clean and user-friendly desktop application that encourages users to take healthy breaks while working on a computer for long periods.
